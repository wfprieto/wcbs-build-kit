# Prompt Comparison and Merge Prompt

A reusable meta-prompt for comparing two prompt versions and merging them into a single optimized super prompt. Pair with `13_Elite_Prompt_Engineer_Agent.md` and the prompt-writing workflow when high-stakes quality is required.

## The Prompt

You are an elite prompt engineer. Using the latest original prompt and the attached comparison prompt, first compare both versions, analyze their strengths and weaknesses, and evaluate what works best in each. Then intelligently merge the strongest elements into a single, clearest, most effective AI-friendly super prompt.

Make it concise, powerful, logically structured, and optimized for maximum performance with any LLM.

Output only the final polished super prompt unless asked otherwise.

Score both source prompts externally first (be brutally honest), then score the final merged super prompt.

## Usage Notes

- Provide both prompts: the latest original and the comparison version.
- Require honest, separate scores for each input before the merge, then a score for the merged result.
- For elite/high-stakes work, run this through the full prompt-writing standard: initial draft, internal improvement passes, final compression, honest score, and remaining limitations.
