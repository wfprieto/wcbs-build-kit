# Archived Repo Publishing Scripts

These scripts were generated for a one-time maintainer-side GitHub publish workflow. They are not part of the portable Super Build Kit install surface and should not be run by ordinary users or downstream agents.

They are archived with non-executable `.archived` names for provenance only. Use the normal kit verification command instead:

```bash
npm run verify
```

Safety status:

- Not required for using the kit.
- Not required for installing skills.
- Not auto-run by any package script.
- Not referenced by `START_HERE`, `LOAD_ORDER`, adapters, skills, or doctor checks.
