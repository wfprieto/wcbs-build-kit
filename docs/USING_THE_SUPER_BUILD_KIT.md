# Using The Super Build Kit

The Super Build Kit is an operating system for agentic software work.

Its operating law is:

**Audit wide. Fix narrow. Prove everything.**

## Daily Use

1. Start with `00_start_here/START_HERE.md`.
2. Confirm the source-of-truth order in `00_start_here/SOURCE_OF_TRUTH.md`.
3. Use `00_start_here/LOAD_ORDER.md` to load only the needed files.
4. Select a tier from `50_audits/AUDIT_TIER_ROUTER.md`.
5. End with the required final report.

## Evidence States

Use only:

- Verified
- Likely
- Suspected
- Unknown
- Not Run
- Blocked

Only Verified evidence may be claimed as confirmed.
